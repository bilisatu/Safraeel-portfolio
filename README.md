# SAFRAEL-PORTFOLIO

A clean, responsive personal portfolio built with semantic HTML, modern CSS, and minimal JavaScript. Suitable for quick customization and deployment on any static host.

## Features
- Responsive layout for mobile and desktop
- Semantic markup and accessible defaults
- No build step; works as plain static files

## Project structure
- `index.html` — Page markup and content
- `style.css` — Styles and responsive rules
- `script.js` — Lightweight client-side interactions
- `LICENSE` — MIT license

## Getting started
Requirements
- A modern web browser
- Optional: Python 3.x (for a local HTTP server) or the VS Code Live Server extension

Local preview (recommended)

```powershell
# From the project root
python -m http.server 5500
# Then open http://localhost:5500 in your browser
```

Alternative
- Open `index.html` directly in your browser, or use VS Code Live Server ("Go Live").
