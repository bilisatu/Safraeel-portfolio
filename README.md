# SAFRAEL-PORTFOLIO

Minimal, production-ready personal portfolio template built with semantic HTML, modern CSS, and vanilla JavaScript.

## Key Features
- Responsive layout for mobile and desktop
- Clean, semantic markup and accessible defaults
- Zero build step — deploy as static files

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)

## Quick Start
Requirements
- Modern web browser
- Optional: Python 3.x (local server) or VS Code Live Server

Local preview (recommended)

```powershell
# From the project root
python -m http.server 5500
# Open http://localhost:5500 in your browser
```

Alternative
- Open `index.html` directly in your browser, or use VS Code Live Server ("Go Live").

## Project Structure
- `index.html` — Page markup and content
- `style.css` — Styles and responsive rules
- `script.js` — Lightweight client-side interactions
- `LICENSE` — MIT license

## Customization
- Modify section content and layout in `index.html` to reflect your profile and projects.
- Adjust colors, typography, and spacing in `style.css` to match your personal branding.
- Extend or refine front-end behavior in `script.js` (for example, animations, interactions, or theme logic).

## Deployment
This project is a static site and can be hosted on any static hosting provider.

- **GitHub Pages**: push the repository, enable Pages in the repository settings, and select the branch that contains `index.html`.
- **Netlify / Vercel**: connect the repository, use the project root as the publish directory, and deploy (no build command required).
